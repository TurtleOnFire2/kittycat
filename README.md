mraow~

## Commands

- `/kc` — opens gui
- `/kc hud` — opens hud editor
- `/kc macros` — opens chat macro menu

## Features

### Misc
- **Bestiary HUD** — Displays info like bestiary gained per hour and an ETA until the next level
- **Chat Macros** — Runs something on configurable messages
- **Pests** — Auto set spawn, auto warp, ESP

### Dungeons
- **Auto LB** — Releases and repulls LB after configurable tick amount
- **Storm** — Auto swap crit item, auto swap armor, auto release LB, auto track storm, auto walk forward, auto swap to term, auto left click with term (you should use a term autoclicker with this, as it only sets the keystate to down…)

### Visual
- **Custom ESP** — `/cesp`
- **Arrow Tracers** — Tracers for your arrows…
